# OrderManV9

## Setup

```bash
git clone https://github.com/keshavsoft/OrderManV9
cd OrderManV9
npm install
```

## Run

```bash
npm start
```

## Build EXE (pkg)

```bash
npx @yao-pkg/pkg app.js --targets node22-win-x64
```

## Run EXE

```bash
OrderManV9.exe
```
